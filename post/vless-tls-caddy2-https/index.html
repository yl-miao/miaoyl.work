VLESS+TLS+HTTP2+Caddy 2与正常https网站共享443端口

2020-10-11
tech

Contents

  • 准备
  • V2ray
    • 安装和更新V2Ray（使用最新的FHS安装）
    • 安装最新发行的geoip.dat 和geosite.dat
    • 配置V2ray服务器
  • 客户端配置
  • 参考

同一台服务器用443端口同时开启正常网站和VLESS服务：443一律连接Caddy 2，用SNI区分不同地址，如果是请求vless则反向代理H2C转发到V2ray服务。

#重装纯净系统
bash <(wget --no-check-certificate -qO- 'https://moeclub.org/attachment/LinuxShell/InstallNET.sh') -d 10 -v 64 -a
#开启BBR
echo net.core.default_qdisc=fq >> /etc/sysctl.conf
echo net.ipv4.tcp_congestion_control=bbr >> /etc/sysctl.conf
sysctl -p
#更新软件
apt update
apt upgrade
apt install wget,curl,sudo,socat,unzip

V2ray

安装和更新V2Ray（使用最新的FHS安装）

1
2

// 安装可执行文件和 .dat 数据文件
# bash <(curl -L https://raw.githubusercontent.com/v2fly/fhs-install-v2ray/master/install-release.sh)

  "log": {
    "loglevel": "warning"

  "inbounds": [
    {
      "port": 2001,
      "listen": "127.0.0.1",
      "protocol": "vless",
      "settings": {
        "clients": [
          {
            "id": "",
            "email": "vless@example.com"
          }
        ],
        "decryption": "none"
      },
      "streamSettings": {
        "security": "none",
        "network": "h2",
        "httpSettings": {
          "path": "/path",
          "host": [
            "vless.example.com"
          ]
        }
      }
    }
  ],
  "outbounds": [
    {
      "tag": "direct",
      "protocol": "freedom",
      "settings": {}
    },
    {
      "tag": "blocked",
      "protocol": "blackhole",
      "settings": {}
    }
  ],
  "routing": {
    "domainStrategy": "AsIs",
    "rules": [
      {
        "type": "field",
        "ip": [
          "geoip:private"
        ],
        "outboundTag": "blocked"
      }
    ]
  }
}

然后重启一下V2ray：systemctl restart v2ray。
